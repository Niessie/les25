Informatie over Html, Css, Javascript en Jquery.

Html

HTML staat voor “Hyper Text Markup Language”.
HTML is een opmaaktaal.
Met opmaaktaal kunt u al een beetje raden wat ermee bedoeld wordt.
HTML beschrijft de opmaak/structuur van een webpagina.

Css

CSS leren is erg handig als je een website wil opmaken/stijlen.
Met HTML kan je alleen de structuur van een website maken en
met CSS zorg je voor het design van de website.

JavaScript

Heel simpel gezegd is JavaScript een populaire scripttaal voor het toevoegen van
interactieve functies en andere dynamische content aan webpagina's.
Bekende voorbeelden van JavaScript content zijn
bijvoorbeeld formulieren, slideshows met foto's en animaties.

Jquery

jQuery is een framework voor JavaScript dat het makkelijker maakt
om interactieve en dynamische websites te bouwen.
